# РуСи89 — двуязычный компилятор C-подобного языка

## Описание проекта

**РуСи89** — это компилятор для C-подобного языка, поддерживающий **двуязычный синтаксис**: можно писать как на английском, так и на русском языке (ключевые слова и идентификаторы).

>   **Статус проекта**: Компилятор находится в разработке. Генерация ассемблерного кода для сложных конструкций (функции с несколькими аргументами, системные вызовы, работа со строками) **ещё в работе**. На данный момент корректно компилируются простые программы с возвратом значений.

---

## Архитектура (6 скриптов)

```
lib.c + test.c → merged.c → tokens.txt → tokens_flat.txt → AST → output.asm → output.elf
```

| Скрипт | Назначение | Статус |
|--------|------------|--------|
| `1.py` | Сшивка файлов, удаление C-комментариев |   Работает |
| `2.py` | Токенизация C-кода (поддержка русских слов) |   Работает |
| `3.py` | Извлечение плоского списка токенов |   Работает |
| `4.py` | Построение AST с отладочным логгированием |   Работает |
| `5.py` | Генерация ассемблерного кода | 🚧 В работе |
| `6.py` | Сборка ELF-файла  | 🚧 В работе |

## Служебные слова, обрабатываемые лексером

### Ключевые слова (управляющие конструкции)

| Русский | Английский | Тип токена |
|---------|------------|------------|
| `если` | `if` | `KW_IF` |
| `иначе` | `else` | `KW_ELSE` |
| `цп` | `while` | `KW_WHILE` |
| `для` | `for` | `KW_FOR` |
| `вернуть` | `return` | `KW_RETURN` |
| `прервать` | `break` | `KW_BREAK` |
| `продолжить` | `continue` | `KW_CONTINUE` |
| `перейти` | `goto` | `KW_GOTO` |
| `выбор` | `switch` | `KW_SWITCH` |
| `случай` | `case` | `KW_CASE` |
| `по_умолчанию` | `default` | `KW_DEFAULT` |

### Типы данных

| Русский | Английский | Тип токена |
|---------|------------|------------|
| `цел` | `int` | `TYPE_INT` / `KW_INT` |
| `овз` | `void` | `TYPE_VOID` / `KW_VOID` |
| `сим` | `char` | `TYPE_CHAR` |
| `беззнак` | `unsigned` | `TYPE_UNSIGNED` |
| `длинный` | `long` | `TYPE_LONG` |
| `короткий` | `short` | `TYPE_SHORT` |
| `вещ` | `float` | `TYPE_FLOAT` |
| `двойной` | `double` | `TYPE_DOUBLE` |

### Составные типы

| Русский | Английский | Тип токена |
|---------|------------|------------|
| `структ` | `struct` | `KW_STRUCT` |
| `объедин` | `union` | `KW_UNION` |
| `перечисл` | `enum` | `KW_ENUM` |
| `размер` | `sizeof` | `KW_SIZEOF` |
| `тип_имя` | `typedef` | `KW_TYPEDEF` |

### Модификаторы

| Русский | Английский | Тип токена |
|---------|------------|------------|
| `статич` | `static` | `MOD_STATIC` |
| `внешн` | `extern` | `MOD_EXTERN` |
| `пост` | `const` | `MOD_CONST` |
| `летуч` | `volatile` | `MOD_VOLATILE` |
| `регистр` | `register` | `MOD_REGISTER` |
| `авто` | `auto` | `MOD_AUTO` |

## Примеры программ

### Пример 1: Только английский синтаксис

```c
int main() {
    return 17;
}
```

### Пример 2: Только русский синтаксис

```c
цел запуск() {
    вернуть 19;
}
```

## Запуск (только для простых программ)

```bash
# 1. Подготовка исходников
echo 'int main() { return 42; }' > test.c
echo '' > lib.c

# 2. Компиляция
python3 1.py          # сшивка
python3 2.py          # токенизация
python3 3.py          # плоские токены
python3 4.py          # AST

# 3. Генерация ASM и сборка
python3 5.py          # → output.asm
python3 6.py          # → output.elf

# 4. Запуск
./output.elf
echo $?               # выведет 42
```
